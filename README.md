# 마실 水 있는 정보 (Water Website)

먹는 물(생수/정수/탄산수 등)의 **종류별 특징**과 **대표 브랜드**를 한눈에 정리하고, 간단한 **물 MBTI 퀴즈(건강/맛 버전)**로 나에게 맞는 물 타입을 추천해주는 React 기반 웹 프로젝트입니다.  
언어는 **한국어/English**를 버튼으로 전환할 수 있습니다.

## 주요 기능

### 1) 물 종류(Types) 정보 탐색
- 물 타입 목록 카드 UI 제공
- 타입 상세 페이지에서 **원천(소스), pH, 미네랄/특징 등 핵심 정보** 확인

### 2) 브랜드(Brands) 탐색
- 브랜드 목록을 **국내/해외**로 구분해 카드로 제공
- 브랜드 상세 페이지에서 브랜드 정보 확인
- 특정 물 타입과 매칭되는 브랜드 추천(결과 페이지에서 활용)

### 3) 물 MBTI 퀴즈 (2가지 버전)
- **건강 버전(Health)**: 건강 목적/선호 기반 질문
- **맛 취향 버전(Taste)**: 맛/감각 선호 기반 질문
- 선택한 답변을 집계하여 **가장 많이 선택된 타입**을 결과로 산출
- 결과 페이지에서 **추천 물 타입 + 추천 브랜드**를 함께 안내

### 4) 홈 화면 히어로 슬라이더
- 홈 상단에서 이미지가 **5초 간격 자동 전환**되는 슬라이더 제공

### 5) 다국어 전환(ko/en)
- 헤더에서 언어 버튼으로 한국어/영어 전환
- 주요 UI/문구/데이터가 언어에 맞게 출력

## 페이지 구성 (Routes)
- `/` : Home (슬라이더 + 진입)
- `/types` : 물 종류 목록
- `/types/:id` : 물 종류 상세
- `/brands` : 브랜드 목록(국내/해외)
- `/brands/:id` : 브랜드 상세
- `/quiz` : 물 MBTI 메뉴(버전 선택)
- `/quiz/health` : 건강 버전 퀴즈
- `/quiz/taste` : 맛 취향 버전 퀴즈
- `/quiz/result/:version/:type` : 결과 페이지(추천 타입/브랜드)
